## 專題題目 (Project Title)  
二手書交易平台 (Second-hand Book Trading Platform)  

## 專案內容 (Project Description)  
本專案旨在建立一個線上二手書交易平台，提供使用者便利的書籍交換與買賣環境。平台主要功能包括：  
1. 使用者帳號註冊與登入  
2. 書籍資訊上架與管理  
3. 書籍搜尋與分類瀏覽  
4. 我的最愛功能
5. 聊天與留言功能，方便買賣雙方溝通
